# Skyma IT – Landing Page

Landing page institucional de **Skyma IT**, una agencia especializada en desarrollo web, automatización y soluciones digitales a medida.

👉 **Demo en vivo:** [skymait.com](https://skymait.com)

## 🧩 Tecnologías utilizadas

- ⚛️ [React](https://reactjs.org/) – Librería para construir interfaces de usuario.
- ⚡ [Vite](https://vitejs.dev/) – Empaquetador ultrarrápido para desarrollo moderno.
- 💅 CSS Modules / TailwindCSS (dependiendo del caso) – Estilado personalizado.
- ✅ ESLint – Control de calidad del código.
- 🚀 [Vercel](https://vercel.com/) – Hosting y despliegue continuo.
